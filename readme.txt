Customize login page with the brand logo and colors with ease.

== Description ==

Admin Tailor plugin lets you customize the wp-login page with a logo, colors and animated background to portray your brands. With easy setup of this plugin enables a website's login page to feel like a genuinely brand website.

This plugin is simple and developed in a performance-priority way.

Major features in Admin Tailor includes:

* Set login page brand logo.
* Set brand color for inputs and button in login page.
* Set animated background & pattern for login page.

== Installation ==

1. Automatic
Go to Dashboard and under Plugins menu click 'Add new' and upload admin-tailor zip file.

OR

2. Manually
Upload it on wp-content/plugins/ directory

Activate it, and set options in 'Admin Tailor' menu under Settings menu in Dashboard.
You're done! And you will see changes on the wp login page.
